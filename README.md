# 📈 Ultra Stock Analytics Terminal

An advanced real-time stock market tracking application built with **Python**, **Streamlit**, and **Plotly**.

## ✨ Features
- **Live Data**: Real-time stock prices via Yahoo Finance.
- **Technical Indicators**: Moving Averages (MA50, MA200) and RSI.
- **Interactive Charts**: Professional Candlestick and Volume charts.
- **Mobile Responsive**: Access via web browser using Ngrok tunnel.

## 🛠️ Tech Stack
- Python
- Streamlit (Web Framework)
- Plotly (Data Visualization)
- yFinance (Financial Data API)

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/ultra-stock-analytics.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run stock_app.py`
